# US099 — `syntek-discount-codes`: Discount Codes & Vouchers

**Last Updated**: 10/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 4 — Backend Feature
Modules

---

## User Story

As a developer, I want a discount codes module so that I can create and manage percentage or
fixed-amount discount codes, gift vouchers, and promotional codes that can be applied at checkout
across all payment providers.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a valid discount code with remaining uses
When validateDiscountCode is called
Then the discount amount is returned

Given a discount code has reached its global usage limit
When another user validates it
Then a validation error is returned

Given a user has already used a code with per_user_limit=1
When the same user validates it again
Then a validation error is returned

Given two requests arrive simultaneously to use the last available code
When both are processed concurrently
Then exactly one succeeds and the other receives a validation error (no overselling)

Given an expired discount code
When validated
Then a validation error with "code expired" reason is returned
```
