## User Story

As a developer, I want a web messaging UI package so that I can add a fully functional live chat
interface — conversation list, chat window, typing indicators, read receipts, presence, and
attachment previews — to any Next.js application, wired to `syntek-messaging` via GraphQL
subscriptions.

## Acceptance Criteria

```gherkin
Given a user opens the messaging panel
When ConversationList renders
Then all conversations are listed with the last message preview, timestamp, unread badge, and
  participant presence dots; conversations are sorted by most recent message

Given a user selects a conversation
When ChatWindow renders
Then the message history loads with infinite scroll (older messages on scroll up); new messages
  arrive in real-time via the messageAdded subscription without polling

Given a user types in MessageInput
When the input is non-empty
Then startTyping mutation is called; TypingIndicator renders the sender name in the chat window
  of other participants; the indicator disappears automatically when stopTyping fires or TTL elapses

Given another participant reads a message
When the readReceiptUpdated subscription fires
Then read receipt ticks update on the sent message without a page reload

Given a message contains a media attachment
When the MessageBubble renders
Then images are rendered inline with lazy loading; other file types show a download card with
  file name and size; video files show a thumbnail with a play button

Given the current user has the admin role in a group conversation
When the GroupChatHeader renders
Then Add Member and Remove Member controls are visible; clicking them calls the corresponding
  mutations with confirmation
```

## Components

| Component              | Description                                                                                                                                             |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MessagingPanel`       | Root two-column layout — ConversationList on left, ChatWindow on right; responsive (collapses to single column on mobile breakpoint)                    |
| `ConversationList`     | Scrollable list with search, unread badges, last message preview, presence dots, and mute indicator                                                     |
| `ConversationItem`     | Individual row — avatar(s), name, last message truncated, timestamp, unread count badge                                                                 |
| `ChatWindow`           | Main chat area — MessageList with infinite scroll, TypingIndicator, MessageInput at bottom                                                              |
| `MessageList`          | Virtualised list (react-window or TanStack Virtual) — groups messages by date; loads older messages on scroll to top                                    |
| `MessageBubble`        | Single message — author, body (plain or Markdown), timestamp, edited indicator, delivery/read ticks, ReactionBar, reply-to preview, attachment previews |
| `ReactionBar`          | Emoji reaction counts with tooltip showing who reacted; click to add/remove own reaction                                                                |
| `MessageInput`         | Textarea with auto-grow, Markdown support (optional), attachment picker, emoji picker, send button                                                      |
| `AttachmentPicker`     | File selector wired to syntek-media-core or syntek-files based on ATTACHMENT_SOURCES config                                                             |
| `AttachmentPreview`    | Inline image, video thumbnail, or file download card                                                                                                    |
| `TypingIndicator`      | Animated "name is typing…" indicator; handles multiple simultaneous typers                                                                              |
| `PresenceDot`          | Online (green) / away (amber) / offline (grey) indicator with tooltip                                                                                   |
| `GroupChatHeader`      | Conversation name, participant count, member management controls for admins                                                                             |
| `NewConversationModal` | User picker to start a direct or group conversation                                                                                                     |
| `MessageSearch`        | Full-text search within a conversation                                                                                                                  |

## Hooks

| Hook                                   | Returns                                                                          |
| -------------------------------------- | -------------------------------------------------------------------------------- |
| `useConversations(options)`            | `{ conversations, loading, error, refetch }`                                     |
| `useMessages(conversationId, options)` | `{ messages, loading, error, fetchOlder, hasMore }` — subscribes to messageAdded |
| `useSendMessage(conversationId)`       | `{ sendMessage, loading, error }` with optimistic append                         |
| `useTyping(conversationId)`            | `{ typingUsers, startTyping, stopTyping }` — debounces startTyping calls         |
| `useReadReceipts(conversationId)`      | `{ markAsRead, readReceipts }` — subscribes to readReceiptUpdated                |
| `usePresence(userIds)`                 | `{ presence }` — subscribes to presenceChanged for given users                   |
| `useConversation(id)`                  | `{ conversation, participants, loading }`                                        |
| `useUnreadCount`                       | `{ totalUnread }` — global unread badge count                                    |
