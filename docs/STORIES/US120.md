# US120 — `@syntek/mobile-feedback`: Mobile Surveys & Feedback Package

**Last Updated**: 10/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 8 — Mobile Packages

---

## User Story

As a developer, I want a React Native surveys and feedback package so that I can render dynamic
survey forms with conditional question logic, NPS prompts, and star ratings on mobile — wired to
`syntek-feedback` via GraphQL and optimised for touch interaction.

## Acceptance Criteria

```gherkin
Given a survey schema is fetched from the API
When MobileSurveyForm renders
Then each question is displayed with the correct touch-optimised input component — star rating,
  NPS row, radio list, checkbox list, text input, boolean switch, or number input

Given a conditional question has its display condition unmet
When the dependent question's answer changes
Then the conditional question is hidden or shown immediately without a network request;
  the FlatList re-renders only the affected items

Given a user answers all required questions and submits
When submitSurveyResponse mutation succeeds
Then a thank-you screen replaces the form and the submit button is permanently disabled

Given the survey spans multiple pages
When the user taps Next
Then required fields on the current page are validated before advancing; a progress bar shows
  the current position

Given MobileNPSWidget is rendered
When the configured displayDelay elapses
Then the widget slides up as a BottomSheet; the user can dismiss it by swiping down; dismissal
  is stored in AsyncStorage so the widget does not reappear in the same session
```

## Components

| Component              | Description                                                                                                         |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `MobileSurveyForm`     | Root ScrollView renderer — maps schema to question components with conditional visibility and multi-page navigation |
| `MobileNPSWidget`      | BottomSheet NPS prompt with configurable delay and AsyncStorage dismissal                                           |
| `MobileSurveyThankYou` | Post-submission confirmation screen                                                                                 |
| `MobileRatingScale`    | Tap-to-rate star or number scale input                                                                              |
| `MobileNPSInput`       | 0–10 horizontal scroll picker with promoter/passive/detractor colouring                                             |
| `MobileSingleChoice`   | RadioButton list with optional "other" text input                                                                   |
| `MobileMultiChoice`    | Checkbox list with optional "other" text input                                                                      |
| `MobileFreeText`       | Multiline TextInput with character counter                                                                          |
| `MobileBooleanToggle`  | Yes/No switch                                                                                                       |
| `MobilePageProgress`   | Progress bar or step indicator for multi-page surveys                                                               |

## Hooks

| Hook                                              | Returns                                                            |
| ------------------------------------------------- | ------------------------------------------------------------------ |
| `useMobileFeedback(surveyId)`                     | `{ schema, loading, error }` — fetches survey schema               |
| `useMobileSurveyResponse(surveyId)`               | `{ submit, loading, error, isSubmitted }`                          |
| `useMobileConditionalVisibility(schema, answers)` | `{ visibleQuestions }` — client-side condition evaluator           |
| `useMobileNPSWidget(surveyId, options)`           | `{ isVisible, dismiss }` — AsyncStorage-backed session suppression |
