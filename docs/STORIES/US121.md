## User Story

As a developer, I want a real-time messaging module so that I can add live chat — direct messages,
group conversations, or context-attached threads (e.g. on an order, a pet profile, or a delivery) —
to any application, with typing indicators, read receipts, presence, and push notifications.

## Acceptance Criteria

```gherkin
Given a user sends a message in a conversation
When the sendMessage mutation completes
Then the message is persisted, a MessageDelivered event is broadcast via Django Channels to all
  active subscribers of that conversation, and a push notification is dispatched via US089 to any
  participant who is not currently subscribed

Given a participant opens a conversation
When the markAsRead mutation is called with the last-read message ID
Then a MessageRead record is created, the read receipt is broadcast to other participants, and
  the conversation's unread count for that participant decrements to zero

Given a user starts typing in a message input
When the startTyping mutation is called
Then a TypingEvent is broadcast via Channels to all other active subscribers; the event is
  not persisted; if stopTyping is not called, the indicator auto-expires after TYPING_TTL_SECONDS

Given a participant's WebSocket connection opens
When the presence group is updated
Then all other active subscribers of shared conversations receive a PresenceChanged event
  with the participant's status (online/away/offline) via Channels

Given CONTEXT_OBJECT_ENABLED is True and a conversation is created with a content_type and object_id
When messages(conversationId) is queried by any participant
Then only participants of that conversation can read messages — cross-conversation leakage
  is blocked at the resolver level

Given an erasure request is processed for a user
When the GDPR erasure hook fires
Then all Message.body fields authored by that user are replaced with ERASURE_PLACEHOLDER,
  the author FK is nulled, and media attachments are unlinked
```

## Configuration

```python
SYNTEK_MESSAGING = {
    # Conversation types
    'ALLOW_DIRECT_MESSAGES': True,
    'ALLOW_GROUP_CONVERSATIONS': True,
    'MAX_GROUP_MEMBERS': 50,
    'ALLOW_CONTEXT_CONVERSATIONS': True,   # attach conversation to any model via generic FK

    # Messages
    'MAX_MESSAGE_LENGTH': 4000,
    'ALLOW_RICH_TEXT': False,              # True = Markdown; stored as Markdown, rendered server-side
    'ALLOW_ATTACHMENTS': True,
    'ATTACHMENT_SOURCES': ['media', 'files'],  # media (US030) | files (US031)
    'MAX_ATTACHMENTS_PER_MESSAGE': 5,

    # Real-time features
    'TYPING_INDICATORS_ENABLED': True,
    'TYPING_TTL_SECONDS': 5,              # auto-clear typing indicator after N seconds
    'PRESENCE_ENABLED': True,
    'PRESENCE_AWAY_TIMEOUT_SECONDS': 120, # mark as away after N seconds of inactivity

    # Reactions
    'REACTIONS_ENABLED': True,
    'REACTION_TYPES': ['👍', '❤️', '😂', '😮', '😢', '👎'],

    # Threads
    'THREAD_REPLIES_ENABLED': False,      # True = allow thread replies on messages

    # Message lifecycle
    'ALLOW_EDIT': True,
    'EDIT_WINDOW_SECONDS': 300,           # 0 = unlimited; edit only within this window
    'ALLOW_DELETE': True,
    'SOFT_DELETE': True,                  # True = replace with "Message deleted"; False = hard delete

    # Notifications
    'PUSH_NOTIFICATIONS_ENABLED': True,   # requires US089
    'PUSH_FOR_OFFLINE_ONLY': True,        # only push if participant is not subscribed via WS

    # Retention & GDPR
    'MESSAGE_RETENTION_DAYS': 0,          # 0 = indefinite
    'ERASURE_PLACEHOLDER': '[message removed]',

    # Moderation
    'MODERATION_ENABLED': False,
    'AUTO_MODERATE_KEYWORDS': [],

    # Tenant
    'TENANT_SCOPED': True,
    'TENANT_CAN_OVERRIDE_MAX_GROUP': True,
}
```

## Models

| Model                     | Key Fields                                                                                                                                                                                                                   |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Conversation`            | `tenant`, `conversation_type` (direct/group/context), `title`, `content_type` (nullable FK), `object_id`, `created_by`, `is_archived`                                                                                        |
| `ConversationParticipant` | `conversation` FK, `user` FK, `role` (member/admin), `joined_at`, `last_read_message` FK, `is_muted`, `is_removed` — unique on `(conversation, user)`                                                                        |
| `Message`                 | `conversation` FK, `author` (nullable FK), `body`, `body_format` (plain/markdown), `status` (sent/delivered/deleted), `parent` FK (thread replies), `edited_at`, `attachments` JSONField — list of `{type, url, name, size}` |
| `MessageReaction`         | `message` FK, `user` FK, `emoji`, unique on `(message, user, emoji)`                                                                                                                                                         |
| `MessageRead`             | `message` FK, `participant` FK, `read_at` — unique on `(message, participant)`                                                                                                                                               |

## GraphQL API

### Queries

| Query                                      | Returns                                                         |
| ------------------------------------------ | --------------------------------------------------------------- |
| `conversations(archived)`                  | Paginated conversation list for current user with unread counts |
| `conversation(id)`                         | Single conversation with participant list                       |
| `messages(conversationId, before, limit)`  | Cursor-paginated messages (newest-first)                        |
| `unreadCount`                              | Total unread message count across all conversations             |
| `conversationParticipants(conversationId)` | Participant list with presence status                           |

### Mutations

| Mutation                                         | Description                                   |
| ------------------------------------------------ | --------------------------------------------- |
| `createConversation(input)`                      | Create direct, group, or context conversation |
| `addParticipant(conversationId, userId)`         | Add participant to group conversation         |
| `removeParticipant(conversationId, userId)`      | Remove participant (admin or self-leave)      |
| `sendMessage(conversationId, body, attachments)` | Send a message                                |
| `editMessage(messageId, body)`                   | Edit own message within EDIT_WINDOW_SECONDS   |
| `deleteMessage(messageId)`                       | Soft or hard delete own message               |
| `markAsRead(conversationId, lastMessageId)`      | Mark conversation as read up to given message |
| `muteConversation(conversationId, muted)`        | Toggle mute for current participant           |
| `addReaction(messageId, emoji)`                  | Add reaction to a message                     |
| `removeReaction(messageId, emoji)`               | Remove own reaction                           |
| `startTyping(conversationId)`                    | Broadcast ephemeral typing indicator          |
| `stopTyping(conversationId)`                     | Clear typing indicator                        |

### Subscriptions (Django Channels)

| Subscription                         | Event                                  |
| ------------------------------------ | -------------------------------------- |
| `messageAdded(conversationId)`       | New message in conversation            |
| `messageUpdated(conversationId)`     | Edit or delete in conversation         |
| `typingChanged(conversationId)`      | Typing indicator start/stop            |
| `presenceChanged`                    | Participant online/away/offline status |
| `readReceiptUpdated(conversationId)` | Read receipt from another participant  |
