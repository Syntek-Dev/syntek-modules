# US097 — `syntek-payments-paypal`: PayPal Adapter

**Last Updated**: 10/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 4 — Backend Feature
Modules

---

## User Story

As a developer, I want a PayPal-specific payments adapter so that I can process PayPal Checkout,
PayPal subscriptions, and handle PayPal webhooks through the syntek-payments-core interface.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given PayPal adapter is configured
When a PayPal order is created
Then an order ID is returned for the front-end to render the PayPal button

Given a user approves a PayPal payment
When the capture mutation is called
Then the order is captured and a Transaction record is created with status succeeded

Given a PayPal webhook arrives
When the signature is verified via PayPal certificate
Then the event is routed and processed

Given a PayPal subscription is created
When the first payment succeeds
Then the Subscription record is activated
```
