## User Story

As a developer, I want a donations UI package so that I can embed donation forms, Gift Aid declarations, and giving history screens that connect to `syntek-donations` via GraphQL.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given the DonationForm is rendered
When a user selects a preset amount and clicks Donate
Then the amount is passed to the payment mutation and the form cannot be submitted again until the first submission resolves

Given a user selects recurring donation
When the donation type toggle changes
Then the form updates to show billing frequency options and the correct mutation is called on submission

Given a user is eligible for Gift Aid
When they check the Gift Aid declaration checkbox
Then the declaration text is displayed in full before submission and the `giftAid: true` flag is included in the mutation payload

Given a user views their giving history
When the GivingHistory component renders
Then all past donations are listed with amount, date, campaign, and Gift Aid status — paginated with infinite scroll
```
