# US047 — `@syntek/layout`: Shell Layout Package

**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 6 — Web Frontend
Packages

---

## User Story

As a developer, I want a shell layout package so that I can compose consistent application shells
with sidebar navigation, top bar, breadcrumbs, and a command palette without rebuilding the layout
for every project.

## MoSCoW Priority

**Must Have**

## Acceptance Criteria

```gherkin
Given the AppShell component is rendered with a nav config
When the page loads
Then the sidebar renders the correct navigation items with active state for the current route

Given a user opens the command palette
When they type a search term
Then matching commands are listed in under 100ms and the selected command executes on Enter

Given the viewport is below the mobile breakpoint
When the layout renders
Then the sidebar collapses to a hamburger menu and the main content fills the full width

Given breadcrumbs are configured for the current route
When the breadcrumb component renders
Then it reflects the current route hierarchy with correct links
```
