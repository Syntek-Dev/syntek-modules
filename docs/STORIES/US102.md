# US102 — `@syntek/ui-invoicing`: Invoice Management UI Package

**Last Updated**: 10/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 7 — Web Frontend
Features

---

## User Story

As a developer, I want an invoicing UI package so that I can embed invoice lists, invoice detail
views with PDF download, credit note management, and online payment via payment links — all wired to
`syntek-invoicing` via GraphQL.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given an invoice list is rendered
When the component loads
Then invoices are displayed with number, date, amount, VAT, status (colour-coded), and a download button — paginated

Given a user clicks the download button on an invoice
When the download completes
Then the invoice PDF is fetched from the GraphQL endpoint and saved to the user's device

Given an invoice has an embedded payment link
When the invoice detail view renders
Then a "Pay Now" button is displayed showing the outstanding amount

Given a user clicks "Pay Now"
When the payment link is opened
Then they are redirected to the payment link checkout (rendered by @syntek/ui-payments)

Given an invoice has status "paid"
When the invoice detail view renders
Then the "Pay Now" button is not shown and a "Paid" badge is displayed

Given a credit note is created against an invoice
When the InvoiceDetail component re-renders
Then the credit note is listed under the invoice with its amount and date, and the outstanding balance is updated
```
