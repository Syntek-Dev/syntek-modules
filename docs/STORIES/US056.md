## User Story

As a developer, I want a multi-step onboarding wizard package so that I can guide new users through setup flows with resumable state, progress tracking, and step validation.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a user begins an onboarding flow
When they complete step 2 and close the browser
Then on returning, the wizard resumes at step 3 with their step 2 data intact

Given a step contains required fields
When the user attempts to advance without completing them
Then an error is shown and navigation to the next step is blocked

Given the user reaches the final step
When they submit
Then the completion mutation fires, the onboarding flag is set on the backend, and the user is redirected to the dashboard

Given a step is marked as skippable in the config
When the user clicks Skip
Then that step is bypassed and the wizard advances to the next required step
```
