# US066 — `syntek-inventory`: Inventory & Stock Management Module

**Status**: To Do
**Epic**: Epic 5 — Backend Compliance & Media

## User Story

As a developer, I want an inventory module so that I can track stock levels across multiple locations, record every movement, and dispatch low-stock alerts automatically.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a stock adjustment is made
When the adjustment is saved
Then a StockMovement record is created with reason code, reference, and the resulting balance — the stock level never goes below zero without an explicit override flag

Given stock falls to or below the configured low-stock threshold for an item
When the threshold check runs after a movement
Then a notification is dispatched via syntek-notifications and a LowStockAlert record is created

Given a stock transfer is made between two locations
When the transfer is processed
Then the source location's stock decreases and the destination's stock increases atomically — no partial state is possible

Given a stock reservation is created on order placement
When the order is cancelled
Then the reserved quantity is released back to available stock
```
