## User Story

As a developer, I want an accounting module so that I can maintain a double-entry ledger, calculate VAT, and sync transactions to Xero, Sage, or QuickBooks Online.

## MoSCoW Priority

**Could Have**

## Acceptance Criteria

```gherkin
Given a transaction is posted to the ledger
When the double-entry validation runs
Then debits equal credits and the transaction is accepted

Given a transaction is posted with debits not equal to credits
When validation runs
Then the transaction is rejected with an `UnbalancedEntryError`

Given Xero credentials are configured
When `sync_to_xero(transaction)` is called
Then the transaction is posted to Xero and the sync status is recorded

Given a VAT period is requested
When the VAT summary is calculated
Then output tax, input tax, and net VAT payable are accurate
```
