## User Story

As a developer, I want a GDPR consent and privacy UI package so that I can present a compliant
cookie consent banner, privacy centre, and data subject request forms that integrate with
`syntek-gdpr` on the backend.

## MoSCoW Priority

**Must Have**

## Acceptance Criteria

```gherkin
Given a user visits the site for the first time
When the page loads
Then the cookie consent banner is shown before any non-essential scripts execute

Given a user accepts only essential cookies
When the consent is saved
Then only essential cookies are set and the consent record is persisted to the backend

Given a user opens the privacy centre
When they submit a Subject Access Request
Then a SAR record is created via the GraphQL mutation and a confirmation email is sent

Given a user withdraws consent
When the withdrawal is saved
Then all non-essential cookies are removed from the browser and the backend consent record is updated
```
