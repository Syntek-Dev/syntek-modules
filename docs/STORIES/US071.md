# US071 — `@syntek/ui-maps`: Maps & Location Picker UI Package

**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 7 — Web Frontend
Features

---

## User Story

As a developer, I want a maps UI package so that I can embed interactive maps, location pickers, and
proximity-sorted location lists that connect to `syntek-locations` via GraphQL.

**Could Have**

## Acceptance Criteria

```gherkin
Given the Map component is rendered
When the tile layer loads
Then the map displays using the provider configured in SYNTEK_MAPS settings — no provider is hardcoded in the component

Given the LocationPicker field is used in a form
When the user types an address and selects a suggestion
Then the selected address and its coordinates are set as the field value and a pin is placed on the map preview

Given a NearbyLocations list is rendered with a reference point
When the component mounts
Then locations are fetched ordered by ascending distance and each result displays name, address, and formatted distance

Given a map has more than 50 markers
When the zoom level is low
Then markers are clustered automatically and the cluster count is displayed on the cluster icon
```
