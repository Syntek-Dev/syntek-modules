## User Story

As a developer, I want a loyalty and referrals module so that I can reward users with points, manage
tier progression, and attribute referrals with accurate commission tracking.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a user completes a qualifying action
When the loyalty engine processes the event
Then the correct points are credited to the user's balance and a LoyaltyTransaction record is created

Given a user's points balance crosses a tier threshold
When tier evaluation runs
Then the user is promoted to the next tier and a notification is dispatched

Given a referral link is clicked and the referee converts
When the conversion event is recorded
Then the referrer receives attribution credit and the configured reward is issued

Given a user redeems points
When the redemption is processed
Then the points are debited, a LoyaltyTransaction is created, and the balance cannot go negative
```
