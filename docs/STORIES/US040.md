# US040 — `syntek-comments`: Threaded Comments Module

**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 5 — Backend
Compliance & Media

---

## User Story

As a developer, I want a comments module so that I can add threaded, moderated comments with
reactions to any model in my application.

## MoSCoW Priority

**Could Have**

## Acceptance Criteria

```gherkin
Given a comment is submitted on a model
When the comment is saved
Then it is linked to the target model via a generic FK and associated with the authenticated user

Given a nested reply is submitted to a comment
When the reply is saved
Then it is linked to the parent comment and depth is tracked

Given a comment contains content matching a configured moderation pattern
When the moderation filter runs
Then the comment is held for review and not displayed until approved

Given a reaction is added to a comment
When the reaction is saved
Then the aggregate reaction counts are updated and the user cannot add the same reaction twice
```
