# US045 — `@syntek/data-hooks`: Data Fetching Hooks

**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 6 — Web Frontend
Packages

## User Story

As a developer, I want a set of data-fetching hooks so that I can implement paginated queries,
infinite scroll, and optimistic mutations across Syntek applications with minimal boilerplate.

## MoSCoW Priority

**Must Have**

## Acceptance Criteria

```gherkin
Given a list query with pagination variables
When `usePaginatedQuery` is called
Then it returns the current page data, page info, and `fetchNextPage` / `fetchPreviousPage` handlers

Given an infinite scroll container
When the user scrolls to the bottom
Then `useInfiniteScroll` triggers the next page fetch automatically without any manual event wiring

Given a mutation is fired with optimistic data
When `useMutation` is called with an optimistic update config
Then the UI updates immediately and rolls back if the mutation fails

Given a query is in flight
When the loading state is checked
Then `isLoading`, `isFetching`, and `error` are accurately reflected throughout the request lifecycle
```
