# US038 — `syntek-email-marketing`: Email Marketing Module

**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 5 — Backend
Compliance & Media

---

## User Story

As a developer, I want an email marketing module so that I can manage mailing lists, send campaigns,
track opens/clicks, and enforce GDPR opt-out.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a contact is on the unsubscribe list
When a campaign is dispatched
Then that contact is excluded from the send — no exception, no error

Given a campaign is sent
When the contact clicks an unsubscribe link
Then they are added to the unsubscribe list and cannot be re-added to any mailing list without a new explicit opt-in

Given a campaign email is sent with an open tracking pixel
When the recipient opens it
Then an `EmailOpen` record is created with timestamp

Given `syntek-gdpr` is installed
When an erasure request is processed for a contact
Then their email address and open/click records are anonymised
```
