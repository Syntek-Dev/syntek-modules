## User Story

As a developer, I want a feedback and surveys UI package so that I can render dynamic survey forms with conditional question logic that connect to `syntek-feedback` via GraphQL.

**Could Have**

## Acceptance Criteria

```gherkin
Given a survey schema is fetched from the API
When the SurveyForm renders
Then each question is displayed with the correct input type — rating scale, NPS widget, free text, or choice selector

Given a conditional question has its display condition unmet
When the dependent question's answer changes
Then the conditional question appears or disappears immediately without a network request

Given a user completes the survey and submits
When the submitSurveyResponse mutation succeeds
Then a thank-you confirmation is shown and the form is replaced — the user cannot resubmit

Given the survey spans multiple pages
When the user advances to the next page
Then the current page's required fields are validated before navigation is permitted
```
