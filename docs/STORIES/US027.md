# US027 — `syntek-donations`: Donation Management Module

**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 4 — Backend Feature
Modules

---

## User Story

As a developer building for a charity or church, I want a donations module so that I can accept
one-off and recurring donations, capture UK Gift Aid declarations, and export HMRC-compatible claim
data.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a donor makes a Gift Aid eligible donation and confirms taxpayer status
When the declaration is saved
Then a `GiftAidDeclaration` record is created linked to the donation with the required HMRC fields

Given Gift Aid declarations are requested for export
When the export runs
Then the output is in HMRC-compatible format with all mandatory fields populated

Given a recurring donation is configured for monthly billing
When the billing date arrives
Then the charge is processed automatically via Stripe Subscriptions without manual intervention

Given a donor chooses to donate anonymously
When the donation is processed
Then no PII is stored and the donation amount is recorded without any donor reference

Given a fundraising campaign has a target amount
When donations are recorded against it
Then the total raised and percentage of target are accurate in real time
```
