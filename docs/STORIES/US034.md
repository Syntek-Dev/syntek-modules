## User Story

As a developer, I want an i18n module so that my application supports multiple languages, locale detection, and UK-standard date/number formatting out of the box.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a user's browser sends `Accept-Language: fr-FR`
When the i18n middleware runs
Then the active locale is set to `fr-FR` for that request

Given a translation key exists in the active locale
When `_("key")` is called
Then the localised string is returned

Given the active locale is `en-GB`
When a date is formatted
Then it uses DD/MM/YYYY format and GBP currency formatting

Given a tenant has a configured default locale
When a user with no language preference makes a request
Then the tenant's default locale is used
```
