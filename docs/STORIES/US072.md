**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 7 — Web Frontend
Features

## User Story

As a developer, I want a scheduling UI package so that I can embed calendar views, availability
grids, and booking forms that connect to `syntek-scheduling` via GraphQL.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given the Calendar component is rendered in month view
When the user navigates to a different month
Then the appointments for the new month are fetched and the calendar updates without a full page reload

Given the AvailabilityGrid is rendered for a resource
When an available slot is clicked
Then the BookingForm opens pre-filled with the selected resource and time slot

Given the BookingForm is submitted
When the bookAppointment mutation succeeds
Then the new appointment appears on the calendar and the slot is no longer shown as available

Given an appointment has `cancelled` status
When it is displayed on the calendar
Then it is rendered with a strikethrough or muted style distinct from confirmed appointments
```
