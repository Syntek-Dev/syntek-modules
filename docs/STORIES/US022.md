# US022 — `syntek-search`: Full-Text Search Module

**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 4 — Backend Feature
Modules

---

## User Story

As a developer, I want a full-text search module so that Django models can be indexed, searched, and
faceted with tenant-isolated indices.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a model is decorated with `@searchable`
When an instance is saved or updated
Then the search index is updated incrementally within 1 second

Given a search query is executed in the context of tenant A
When the search engine runs
Then results from tenant B's index are never returned

Given a search query includes facet aggregations
When the response is returned
Then facet counts are accurate and match the filtered result set

Given autocomplete is enabled for a text field
When a partial string of 2+ characters is submitted
Then prefix and fuzzy suggestions are returned within 100ms

Given `BACKEND: opensearch` is configured
When the module initialises
Then it connects to OpenSearch instead of Elasticsearch with no code changes
```
