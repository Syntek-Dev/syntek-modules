# US036 — `syntek-geo`: Address & Geo Module

**Last Updated**: 06/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 5 — Backend
Compliance & Media

---

## User Story

As a developer, I want a geo module so that I can look up UK postcodes, validate addresses, and
geocode locations.

## MoSCoW Priority

**Could Have**

## Acceptance Criteria

```gherkin
Given a valid UK postcode is submitted
When `lookup_postcode(postcode)` is called
Then it returns the full address (street, town, county) and coordinates (lat/lng)

Given an invalid UK postcode is submitted
When `lookup_postcode(postcode)` is called
Then it raises `InvalidPostcodeError`

Given an address string is submitted for geocoding
When `geocode(address)` is called
Then it returns latitude and longitude coordinates
```
