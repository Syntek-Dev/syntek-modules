# US031 — `syntek-files`: MinIO File Storage & Document Management

**Last Updated**: 10/03/2026 **Version**: 0.2.0 **Status**: To Do **Epic**: Epic 5 — Backend

## User Story

As a developer, I want a file storage module so that PDFs and office documents are stored in MinIO
with presigned URL access, file versioning, permission-gated download, and type-safe upload
validation — completely separate from media (images/video) which lives in Cloudinary.

## Acceptance Criteria

```gherkin
Given a PDF is uploaded via the uploadFile mutation
When the upload handler runs
Then the file is stored in MinIO and a File record is created in PostgreSQL with bucket, object_key,
filename, size, mime_type, version, tenant FK, and uploaded_by

Given a user with the required permission requests a download URL
When the presigned URL is generated
Then it expires after DOWNLOAD_URL_TTL seconds (default 900)

Given a user without the required permission requests a download URL
When the documentUrl query is resolved
Then a 403 is returned and no presigned URL is generated

Given an image MIME type (JPEG, PNG) is submitted to the uploadFile mutation
When the MIME type is validated
Then the upload is rejected with 422 and a clear error message directing the caller to syntek-media

Given a file is uploaded when VERSIONING_ENABLED is True
When a second upload with the same logical key occurs
Then a new FileVersion record is created; the previous version remains accessible and is not deleted

Given a file is deleted
When the deletion handler runs
Then the MinIO object is removed AND the File record (and all FileVersions) are removed from
PostgreSQL atomically

Given FILE_RETENTION_DAYS is set to 90
When a File record's created_at exceeds the retention window
Then a periodic Celery task marks the file as expired and queues it for deletion
```

## Configuration

```python
SYNTEK_FILES = {
    # --- MinIO credentials (required) ---
    'MINIO_ENDPOINT': env('MINIO_ENDPOINT'),          # e.g. 'minio.example.com:9000'
    'MINIO_ACCESS_KEY': env('MINIO_ACCESS_KEY'),
    'MINIO_SECRET_KEY': env('MINIO_SECRET_KEY'),
    'MINIO_BUCKET': env('MINIO_BUCKET', default='files'),
    'MINIO_SECURE': True,                             # Use HTTPS for MinIO connection

    # --- Upload ---
    'ALLOWED_MIME_TYPES': [                           # Explicit allowlist — no images, no executables
        'application/pdf',
        'application/msword',
        'application/vnd.openxmlformats-officedocument.wordprocessingml.document',  # .docx
        'application/vnd.ms-excel',
        'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet',        # .xlsx
        'application/vnd.ms-powerpoint',
        'application/vnd.openxmlformats-officedocument.presentationml.presentation', # .pptx
        'application/vnd.oasis.opendocument.text',    # .odt
        'application/vnd.oasis.opendocument.spreadsheet',  # .ods
        'text/csv',
        'text/plain',
    ],
    'MAX_FILE_SIZE_MB': 100,
    'OBJECT_KEY_STRATEGY': 'uuid',                   # 'uuid' | 'path' — object key generation

    # --- Access ---
    'DOWNLOAD_URL_TTL': 900,                         # Presigned URL expiry (seconds)
    'REQUIRE_PERMISSION': True,                       # Gate downloads behind syntek-permissions
    'DOWNLOAD_PERMISSION': 'syntek_files.download_file',

    # --- Versioning ---
    'VERSIONING_ENABLED': True,                      # Keep all versions on re-upload
    'MAX_VERSIONS_PER_FILE': 10,                     # 0 = unlimited
    'RETENTION_DAYS': 0,                             # 0 = keep forever; > 0 = auto-expire

    # --- Tenancy ---
    'TENANT_SCOPED': True,                           # Prefix all object keys with tenant ID
    'TENANT_BUCKET_STRATEGY': 'prefix',             # 'prefix' | 'bucket' (one bucket per tenant)
}
```

## Models

| Model           | Field             | Notes                                             |
| --------------- | ----------------- | ------------------------------------------------- |
| `File`          | `id`              | UUID primary key                                  |
|                 | `bucket`          | MinIO bucket name                                 |
|                 | `object_key`      | Full object key in MinIO                          |
|                 | `filename`        | Original filename as uploaded                     |
|                 | `size`            | Bytes                                             |
|                 | `mime_type`       | Validated MIME type                               |
|                 | `current_version` | FK to latest `FileVersion`                        |
|                 | `tenant`          | FK — tenant isolation                             |
|                 | `uploaded_by`     | FK — user who uploaded                            |
|                 | `created_at`      | Timestamp of first upload                         |
|                 | `expires_at`      | Nullable — set by retention policy                |
| `FileVersion`   | `file`            | FK to `File`                                      |
|                 | `version_number`  | Auto-incrementing integer per file                |
|                 | `object_key`      | MinIO key for this version (may differ from File) |
|                 | `size`            | Bytes for this version                            |
|                 | `uploaded_by`     | FK — who uploaded this version                    |
|                 | `created_at`      | Timestamp                                         |
|                 | `checksum`        | SHA-256 hex digest of the file content            |
| `FileAccessLog` | `file`, `user`    | Audit log of presigned URL generation events      |

## GraphQL API

### Mutations

| Mutation                        | Description                                             |
| ------------------------------- | ------------------------------------------------------- |
| `uploadFile(file, filename)`    | Upload to MinIO, create File + FileVersion, return File |
| `deleteFile(id)`                | Atomic MinIO + DB deletion (all versions)               |
| `deleteFileVersion(versionId)`  | Remove a specific version from MinIO + DB               |
| `restoreFileVersion(versionId)` | Set a previous version as current_version               |

### Queries

| Query                             | Returns                                              |
| --------------------------------- | ---------------------------------------------------- |
| `files(page, pageSize, mimeType)` | Paginated `[File]` — tenant-scoped, permission-gated |
| `file(id)`                        | Single `File` with current version metadata          |
| `fileUrl(id)`                     | Pre-signed download URL for current version          |
| `fileVersions(fileId)`            | `[FileVersion]` — all versions for a file            |
| `fileVersionUrl(versionId)`       | Pre-signed download URL for a specific version       |
