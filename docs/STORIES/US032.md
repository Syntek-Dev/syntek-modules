## User Story

As a developer building a membership organisation platform, I want a membership module so that I can
define tiers, manage renewals, and provide a queryable member directory.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a membership expires in 30 days
When the renewal reminder Celery task runs
Then the member receives a renewal notification via `syntek-notifications`

Given a membership lapses without renewal
When the lapse Celery task runs
Then the member's tier permissions are automatically revoked via `syntek-permissions`

Given a member directory query is made
When the resolver runs
Then only members who have opted into directory visibility are returned

Given a membership tier is upgraded
When the upgrade is processed
Then the old tier permissions are revoked and new tier permissions are granted atomically

Given a membership is renewed
When the renewal payment succeeds
Then the membership expiry date is extended by one period from the current expiry (not from today)
```
