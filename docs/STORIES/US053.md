## User Story

As a developer, I want a search UI package so that I can add a search bar with autocomplete, faceted filters, and result highlighting that connects to `syntek-search` via GraphQL.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a user types in the search bar
When the input is debounced (300ms)
Then autocomplete suggestions are fetched and displayed in a dropdown

Given the user submits a search query
When the results page renders
Then matching terms in result titles and excerpts are highlighted

Given facet options are returned with the search results
When the facets panel renders
Then each facet shows its option labels with result counts and active filters are visually indicated

Given a facet filter is applied
When the filter state changes
Then the search re-fetches with the updated filter variables and the URL query string reflects the active filters
```
