# US054 — `@syntek/ui-reporting`: Charts & Report Builder UI

**Last Updated**: 06/03/2026

**Status**: To Do
**Epic**: Epic 6 — Web Frontend Packages

---

## User Story

As a developer, I want a reporting UI package so that I can display charts, build reports, and trigger exports that connect to `syntek-reporting` via GraphQL.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a line chart is rendered with time-series data
When the data updates
Then the chart re-renders with the new data and maintains the correct axis labels and scale

Given a user uses the report builder
When they select metrics and grouping
Then a preview query is fired and the chart updates to reflect the current configuration

Given a user triggers a CSV export from the report UI
When the export mutation completes
Then a download link is presented and the file contains the data currently shown in the report

Given a report is displayed
When a user without the required role views the page
Then the report data is not rendered and a permission denied message is shown
```
