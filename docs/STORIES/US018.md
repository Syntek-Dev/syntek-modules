# US018 — `syntek-groups`: Groups & Organisational Hierarchy Module

## User Story

As a developer, I want a groups module so that users can be organised into nested teams with role-scoped membership and email-based invitation flows.

## MoSCoW Priority

**Should Have**

## Acceptance Criteria

```gherkin
Given a group hierarchy Organisation > Department > Team
When a user is a member of Team
Then they are queryable as a descendant of Organisation without a cross-tenant query

Given a user is invited to a group via email
When they accept the invitation within the expiry window
Then they are added as a member with the specified role

Given an invitation has expired
When the invitee attempts to accept it
Then they receive an `InvitationExpiredError` and must request a new invitation

Given a permission bundle is assigned to a group
When a member of the group makes a request
Then they inherit all permissions in the bundle

Given the configured maximum group depth is 5
When an attempt is made to nest a group at depth 6
Then a `MaxDepthExceededError` is raised
```
