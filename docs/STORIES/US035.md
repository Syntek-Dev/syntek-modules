# US035 — `syntek-caldav`: CalDav Integration Module

**Last Updated**: 06/03/2026
**Version**: 0.1.0
**Status**: To Do
**Epic**: Epic 5 — Backend Compliance & Media

## User Story

As a developer, I want a CalDav client module so that I can sync calendar events with a Radicale CalDav server hosted on `syntek-infrastructure`.

## MoSCoW Priority

**Could Have**

## Acceptance Criteria

```gherkin
Given CalDav credentials are configured
When `sync_calendar(tenant)` is called
Then events from the CalDav server are fetched and stored in the local `CalendarEvent` model

Given a local event is created
When the sync task runs
Then the event is pushed to the CalDav server

Given the CalDav server is unreachable
When a sync is attempted
Then the failure is logged and the task is retried via `syntek-tasks` without raising an uncaught exception
```
