# US123 — `@syntek/mobile-messaging`: Mobile Messaging & Live Chat Package

**Last Updated**: 10/03/2026 **Version**: 0.1.0 **Status**: To Do **Epic**: Epic 8 — Mobile Packages

---

## User Story

As a developer, I want a React Native messaging package so that I can add a fully functional live
chat interface to any mobile app — conversation list, chat screen, typing indicators, read receipts,
presence, camera/media attachments, and push notifications — wired to `syntek-messaging` via GraphQL
subscriptions.

## Acceptance Criteria

```gherkin
Given a user opens the chat tab
When MobileConversationList renders
Then all conversations are shown in a FlatList with last message preview, unread badge, presence
  dot, and pull-to-refresh; tapping a conversation navigates to MobileChatScreen

Given a user is in a chat screen
When new messages arrive via the messageAdded subscription
Then they appear at the bottom of the inverted FlatList in real-time without a manual refresh;
  the user does not need to scroll up to trigger loading

Given a user taps the attachment button
When MobileAttachmentSheet opens
Then they can choose camera, photo library, or file picker; selected media is uploaded via
  syntek-media-core (US030) or syntek-files (US031); a progress indicator is shown during upload

Given the app is backgrounded and a new message arrives
When the push notification is received
Then tapping it deep-links directly to the correct MobileChatScreen with the conversation loaded

Given a user long-presses a message
When the MobileMessageActionSheet opens
Then available actions are shown (react, reply, copy, edit own, delete own); tapping React opens
  an emoji picker row; the selected reaction is toggled via addReaction/removeReaction

Given another participant reads a message
When the readReceiptUpdated subscription fires
Then delivery/read ticks on sent messages update in real-time
```

## Components

| Component                     | Description                                                                                                                |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| `MobileConversationList`      | FlatList of conversations with pull-to-refresh, search bar, unread badges, and presence dots                               |
| `MobileConversationItem`      | Row — avatar(s), name, last message preview, timestamp, unread badge                                                       |
| `MobileChatScreen`            | Full-screen chat — inverted FlatList for messages, MessageInput pinned at bottom, keyboard-avoiding                        |
| `MobileMessageBubble`         | Single message — body, timestamp, delivery/read ticks, reaction row, attachment previews; long-press triggers action sheet |
| `MobileMessageActionSheet`    | BottomSheet with contextual actions (react, reply, copy, edit, delete)                                                     |
| `MobileEmojiPicker`           | Horizontally scrolling emoji reaction selector                                                                             |
| `MobileMessageInput`          | Auto-growing TextInput with send button, attachment button, and (optional) emoji button                                    |
| `MobileAttachmentSheet`       | BottomSheet for camera / photo library / file picker selection                                                             |
| `MobileAttachmentPreview`     | Inline image thumbnail, video thumbnail with play, or file card within message bubble                                      |
| `MobileTypingIndicator`       | Animated "…" dots with sender name                                                                                         |
| `MobilePresenceDot`           | Coloured presence indicator (online/away/offline)                                                                          |
| `MobileNewConversationScreen` | User search and participant picker to start a new chat                                                                     |
| `MobileGroupInfoScreen`       | Group name, member list with add/remove (admins)                                                                           |

## Hooks

| Hook                                         | Returns                                                                   |
| -------------------------------------------- | ------------------------------------------------------------------------- |
| `useMobileConversations(options)`            | `{ conversations, loading, error, refetch }`                              |
| `useMobileMessages(conversationId, options)` | `{ messages, loading, fetchOlder, hasMore }` — subscribes to messageAdded |
| `useMobileSendMessage(conversationId)`       | `{ sendMessage, loading, error }` with optimistic append                  |
| `useMobileTyping(conversationId)`            | `{ typingUsers, startTyping, stopTyping }`                                |
| `useMobileReadReceipts(conversationId)`      | `{ markAsRead }` — auto-marks on screen focus                             |
| `useMobilePresence(userIds)`                 | `{ presence }`                                                            |
| `useMobileUnreadCount`                       | `{ totalUnread }` — drives tab badge                                      |
